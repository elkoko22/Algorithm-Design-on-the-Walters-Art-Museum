Welcome

This website is a summary of a group project and what is posted here describes briefly the main features of the project. The project itself entails curating a dataset and designing an algorithm to process that dataset in a meaningful way. For this assignment, our group has chosen to work with data from the “Walters Art Museum” in Baltimore, Maryland. From the very beginning, it was decided to focus on a dataset that covers a common interest in art. So, searching GitHub, the team concluded to Walters Art Museum collections.

To learn more about the chosen datasets, how the team model and curate the data, the choices in algorithm design and the team’s labor division, please see the related pages from the top menu.
